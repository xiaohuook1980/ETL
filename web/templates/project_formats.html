什么是 format？ 一个 format = 一种数据结构（如"白班计时考勤"/"计件考勤"/"横向月度考勤"）。同项目下不同结构的数据可独立配置全套规则（特征列/列映射/企业归属/项目归属/有效性/pivot）。
每个维度（考/账/发/工）下可有 N 个 format。规则按 format 隔离，dispatcher 按 format_id 加载。
