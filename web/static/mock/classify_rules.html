列映射 文件列 → mart.attendance 字段（每条规则独立配置）
规则：
mart 字段 ← 文件列名（支持 * 通配；空=不映射） 说明
日期 shift_date * ← 必填
姓名 name_raw * ← 必填
工时 hours ⚪ ← ⚪ hours / quantity 二选一
件数 quantity ⚪ ← 计件项目用
岗位 worker_type ← 如 装卸工/分拣员/司机
部门 floor_or_group ← 如 A区仓/餐务部
类型 worker_class ← 临时工 / 长期工
班次 shift_name ← 白班 / 夜班 / ...
身份证 id_card_raw ← 进 attendance 行（冗余）
手机号 mobile ← 进 workers 表，用作工人匹配
备注 extra_data ← 多列；全装 JSON（key=列名 value=单元格值）
必填：日期 + 姓名 + (工时 或 件数)。
specialized handler 的规则（双区域 / 长期工 pivot 等）：列映射由代码层处理，本表不生效。

勾选语义：行首=整条规则启用；特征列前=该字段是否参与匹配（不勾=对所有 sheet 生效，多数考勤场景）；过滤列前=该字段是否参与匹配。
